# Excel-Dashboard-with-AI

Adidas Sales Dashboard (Excel + VBA)

This project contains an automated Adidas Sales Dashboard built in Microsoft Excel using VBA.
The dashboard shows sales trends, units sold, profit, category-wise distribution, region-wise performance, and retailer analysis.
